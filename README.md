# Scientific Calculator

A free web-based scientific calculator with:

- Arithmetic (+, -, *, /, %)
- Power, square root, factorial
- Trigonometry (sin, cos, tan, csc, sec, cot)
- Logarithms (log10, ln)
- Absolute, ceiling, floor
- Permutation (nPr) and Combination (nCr)
